Laser Security System

Aim: Protect the house using laser security. Laser interruption triggers alarm and LED.

Working: Laser reflects through 3 mirrors to LDR sensor. Beam interruption triggers buzzer, LED, and OLED alert.

Components: Laser, LDR Sensor, BC547 Transistor, LED, Buzzer, Mirrors

Advantages: Low cost, high security, continuous monitoring.

Future Scope: IoT alerts, mobile notifications.

Securing homes, securing futures.

Smart Irrigation System

Aim: Automatic watering based on soil moisture.

Working: Soil Moisture Sensor detects dry/wet soil. Pump and LED activate accordingly. OLED displays real-time soil status.

Components: Soil Moisture Sensor, Water Pump, Relay, LED

Advantages: Water saving, efficient, reduces human effort.

Future Scope: HVAC or IoT environmental automation.

Every drop counts towards a greener planet.

Flame, Gas, and Smoke Detection

Aim: Detect fire, smoke, or gas leakage.

Working: MQ-2 detects gas/smoke, LDR detects flame. Buzzer, LED, and OLED alert trigger on detection.

Components: MQ-2 Sensor, LDR Sensor, Buzzer, LED

Advantages: Early warning, automatic alerts.

Future Scope: Ventilation/fire suppression, mobile alerts.

Safety first, always alert.

Temperature & Humidity Monitoring

Aim: Real-time monitoring of environmental conditions.

Working: DHT11 sensor measures temperature and humidity. OLED displays readings.

Components: DHT11 Sensor, OLED Display

Advantages: Useful for greenhouses, homes, labs.

Future Scope: IoT integration, mobile notifications, weather prediction.

Know your environment, control your comfort.

Renewable Energy Solutions

Aim: Generate energy using wind and mechanical sources.

Working: Windmill powers LED, seesaw dynamo triggers buzzer. Demonstrates renewable energy practically.

Components: Windmill, Dynamo Motor, LED, Buzzer

Advantages: Educational, environmentally friendly.

Future Scope: Power small appliances, energy storage integration.

Powering smarter, greener futures.

Arduino Code

Below is the code for Smart Home and Eco-Friendly Energy Solutions:

#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>
#include "DHT.h"

// --- Your Arduino code goes here ---
#define DHTPIN 2
#define DHTTYPE DHT11
DHT dht(DHTPIN, DHTTYPE);

Adafruit_SSD1306 d(128, 64, &Wire);

int soil = A0, sLED = 7, flame = 8, fLED = 13, smoke = 4, smLED = 3, buzz = 11;
int s = A1, l = 6;      // rain pin + rain LED

unsigned long t = 0;
int SOIL_T = 500, RAIN_T = 650;

int relayPump = 10;

void alert(const char* l1, const char* l2 = "") {
  d.clearDisplay();
  d.setTextColor(1);
  d.setTextSize(2);
  d.setCursor(0, 14);
  d.print(l1);
  if (l2[0]) { d.setCursor(0, 36); d.print(l2); }
  d.display();
}

void setup() {
  pinMode(sLED, OUTPUT); pinMode(fLED, OUTPUT); pinMode(smLED, OUTPUT); pinMode(buzz, OUTPUT);
  pinMode(flame, INPUT); pinMode(smoke, INPUT); pinMode(l, OUTPUT);
  pinMode(relayPump, OUTPUT); digitalWrite(relayPump, LOW);
  d.begin(SSD1306_SWITCHCAPVCC, 0x3C); d.clearDisplay(); d.display();
  dht.begin();
}

void loop() {
  int dry = analogRead(soil) > SOIL_T;
  int f = !digitalRead(flame);
  int sm = !digitalRead(smoke);
  int r = analogRead(s) < RAIN_T;

  digitalWrite(sLED, dry); digitalWrite(fLED, f); digitalWrite(smLED, sm); digitalWrite(l, r);

  if (dry) digitalWrite(relayPump, LOW); else digitalWrite(relayPump, HIGH);

  if (f || sm) { digitalWrite(buzz, HIGH); t = 0; }
  else if (r) {
    if (!t) t = millis();
    if (millis() - t < 5000) digitalWrite(buzz, ((millis() - t) % 100) < 50);
    else digitalWrite(buzz, LOW);
  } else { digitalWrite(buzz, LOW); t = 0; }

  if (f) { alert("   FLAME!"); return; }
  if (sm) { alert("SMOKE /   GAS LEAK!"); return; }
  if (r) { alert("RAIN! TAKE CLOTHES"); return; }

  float temp = dht.readTemperature(); float hum = dht.readHumidity();

  d.clearDisplay(); d.setTextColor(1); d.setTextSize(1);
  d.setCursor(0, 0); d.print("Temp: "); d.print(temp); d.print(" C");
  d.setCursor(0, 14); d.print("Hum: "); d.print(hum); d.print(" %");
  d.setCursor(0, 28); d.print("Soil: "); d.print(dry ? "DRY" : "WET");
  if (dry) { d.setCursor(0, 42); d.print("Water Needed"); }
  d.display();
}
